## **Project Overview**
Happy Tails is a platform that integrates pet adoption, pet sitting, and AI-powered care advice into a single, secure application. The goal is to simplify pet care management for owners, sitters, and shelters while fostering responsible pet ownership.

---

## **Features**
- **Pet Adoption:**
  - Browse adoptable pets with filters (breed, age, location).
  - Submit adoption applications.
  - (Future release possibility) Use the Adoption Readiness Simulator to prepare for pet ownership.
- **Pet Sitting:**
  - Find and book trusted pet sitters.
  - Manage availability and services.
- **AI Care Assistant:**
  - Receive personalized care reminders and advice.
  - Track pet health and vaccination records.
- **Administrator Tools:**
  - Manage user accounts and roles.
  - Monitor AI performance and flag inaccuracies.

## **Setup**
### **Prerequisites**
- Node.js (v16 or later)
- npm or yarn

### **Installation**
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/happy-tails.git
   cd happy-tails
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

---

## **Running the Project**
### **Development Mode**
Start the development server:
```bash
npm run dev
```
Your app will be available at `http://localhost:5173`.

### **Production Build**
Create a production build:
```bash
npm run build
```
Preview the production build:
```bash
npm run preview
```
